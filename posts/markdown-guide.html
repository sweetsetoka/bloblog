Markdown 사용법

2026-02-02

블로그 글 작성에 유용한 Markdown 문법을 정리했습니다.

제목

# 기호를 사용해서 제목을 만들 수 있습니다. #의 개수에 따라 제목 크기가 달라집니다.

강조

텍스트를 **굵게** 또는 *기울임*으로 강조할 수 있습니다.

코드

인라인 코드는 백틱(`)으로 감싸면 됩니다. 코드 블록은 다음과 같이 작성합니다:

function hello() {
  console.log("Hello, World!");
}

링크와 이미지

링크는 [텍스트](URL) 형식으로, 이미지는 ![대체텍스트](이미지URL) 형식으로 추가합니다.

Markdown을 잘 활용하면 빠르고 깔끔하게 글을 작성할 수 있습니다.
